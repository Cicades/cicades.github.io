css书写规范

  • 属性的书写顺序

    Formatting Model(布局方式、位置) > Box Model（尺寸） > Typographic（文本相关） > visual（视觉效果）

    .selector {
    /*fomatting model*/
    position: absolute;
    top: 50px;
    left: 0;
    overflow-x: hidden;

    /*box size model*/
    width: 200px;
    padding: 20px;
    border: 1px solid #ccc;

    /*typographic*/
    font-size: 14px;
    line-height: 20px;

    /*visual*/
    background: #fff;
    color: #333;
    transition: all 1s;
}
