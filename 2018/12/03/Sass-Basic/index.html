Sass Basic

By Cicades at: 2018-12-03
Sass

編譯sass

  • 單文件轉換——sass <inputFileName> <outputFileName>

  • 單文件監聽命令——sass --watch <inputFileName>:<outputFileName>

    對sass或者scss文件進行修改，編譯器會自動將其編譯為css文件，對原有文件進行跟新

  • 目錄監聽——sass --watch <inputDir/sass>:<outputDir/stylesheets>

  • 編譯排版配置選項——--style <options>

    1. nested——以縮進格式排版
    2. expanded——正常格式排版
    3. compact——壓縮排版
    4. compressed——壓縮成一行排版

  • 文件保存——style.scss與style.sass之間的語法存在差異，後者采用縮進和換行進行標識代碼段落和語句結束。

基本語法

  • 變量

    1. 聲明變量——$varName:value,可以用空格或者逗號分割為變量賦予多個屬性值
    2. 變量作用於——聲明在大括號内的變量只能在大括號内使用

  • 嵌套

    //原生css
#content article h1 { color: #333 }
#content article p { margin-bottom: 1.4em }
#content aside { background-color: #EEE }

    1. 父選擇器標識——&（相當於js中的this）

    2. 群組選擇器嵌套

      //原生css
.container h1, .container h2, .container h3 { margin-bottom: .8em }
//sass
.container {
  h1, h2, h3 {margin-bottom: .8em}
}

    3. 子組合選擇器和同層選擇器：> 、+、~

      //sass
article {
  ~ article { border-top: 1px dashed #ccc }
  > section { background: #eee }
  dl > {
    dt { color: #333 }
    dd { color: #555 }
  }
  nav + & { margin-top: 0 }
}
//css
article ~ article { border-top: 1px dashed #ccc }
article > footer { background: #eee }
article dl > dt { color: #333 }
article dl > dd { color: #555 }
nav + article { margin-top: 0 }

    4. 嵌套屬性

      //原生css
nav {
  border-style: solid;
  border-width: 1px;
  border-color: #ccc;
}
//sass
nav {
  border:{
  style: solid;
  width: 1px;
  color: #ccc;
  }
}

    導入sass文件

    1. 語法——@import "<filename>";

    2. 默認變量值

      $fancybox-width: 400px !default;
.fancybox {
    width: $fancybox-width;
}

    混合器

    1. 定義—— @mixin <name> {<content>}

    2. 調用——@include <mixinName>

    3. 混合器傳參

      //定義混合器
@mixin link-colors($normal, $hover, $visited) {
  color: $normal;
  &:hover { color: $hover; }
  &:visited { color: $visited; }
}
//引用混合器
@include link-colors(blue,red,green)
//或者
 @include link-colors(
      $normal: blue,
      $visited: green,
      $hover: red
  );
  //為混合器設置默認參數
  @mixin link-colors($normal,$hover: $normal,$visited: $normal){//$hover,$visited的默認參數爲$normal
  color: $normal;
  &:hover { color: $hover; }
  &:visited { color: $visited; }
}

選擇器繼承

繼承的選擇器不要使用後代選擇器，這樣會使編譯後的css中的結構變得很複雜

  1. 語法——@extend <selector>

  2. 原理

    @extend背后最基本的想法是，如果.seriousError @extend .error， 那么样式表中的任何一处.error都用.error.seriousError这一选择器组进行替换。这就意味着相关样式会如预期那样应用到.error和.seriousError。

SassScript

  1. 变量定义

    $width: 1px

  2. 数据类型（6种）

    • 数字（1，2，13，10px）
    • 子符串(有引号字符串，无引号字符串)
    • 颜色(blue,#ccc,rgba(255,0,0,0.5))
    • 布尔型 (true,false)
    • 空值（null）
    • 数组（list）
    • maps 相当于JavaScript中的objec，（key1：value1, key2: value2）

  3. 控制指令（控制语句）

    • @if
    p{
    @if 1 + 1 == 2 {border: 1px solid #ccc}
    @if 5 < 3 {border: 2px dotted}
    @if null {border: 3px double}
}
//结果 p{border: 1px solid}

    • @for

      @for $i from 1 through 3 {
   .item-#{$si} {width: 2em * $i;}
}

    • @each

      @each $item in $list {
   ...code
}
//多重变量
@each $item1, $item2, $item3 in (1 2 3), (4, 5, 6), (7, 8, 9){
   $item1 + $item2 + $item3;
}
//可用上述方法，遍历maps的键和值

    • 参数变量（类似es6中rest变量）

      //扩展运算符`...`
@mixin box-shadow($shadows...) {
 -moz-box-shadow: $shadows;
 -webkit-box-shadow: $shadows;
 box-shadow: $shadows;
}
.shadows {
 @include box-shadow(0px 4px 5px #666, 2px 6px 10px #999);
}

    • 函数定义@function

      @function fooname (){
   @return "hello world";//@return 定义函数的返回值
}

    sass常用函数

    • 数组函数
      1. lenght($list)——返回数组的长度
      2. nth($list,$index)——返回指定数组的第几个元素，下表是从1开始的
